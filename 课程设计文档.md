# 地址簿管理器 —— 课程设计文档

## 一、项目简介

本项目是面向对象程序设计课程设计中的「地址簿管理器」，使用 **Java Swing** 编写图形界面，用来管理联系人的姓名、电话、邮件等信息。数据通过 **Java 对象流**（序列化）保存到文件中。

开发环境：JDK 8+，只用 Java 标准库，不需要任何第三方 jar 包。

---

## 三、各类说明

### 3.1 Person.java —— 人员类

存储联系人的基本信息，实现了 `Serializable` 接口，这样就可以用对象流读写。

**成员变量：**

| 变量     | 类型   | 说明               |
| -------- | ------ | ------------------ |
| category | String | 类别（朋友/家人等）|
| name     | String | 姓名               |
| phone    | String | 电话号码           |
| email    | String | 邮件地址           |

**主要方法：**
- 构造方法：初始化四个属性
- getter / setter：各属性的读写方法
- `changeCategory(String)`：更改类别

### 3.2 AddressBook.java —— 数据管理类

用 `ArrayList<Person>` 存储所有联系人，提供增删改查操作，以及用对象流读写文件。

**主要方法：**

| 方法                       | 说明                   |
| -------------------------- | ---------------------- |
| `add(Person)`              | 添加一个联系人         |
| `delete(int)`              | 删除指定位置的联系人   |
| `update(int, Person)`       | 修改联系人信息         |
| `move(int, int)`           | 移动联系人的位置       |
| `get(int)`                 | 获取指定位置的联系人   |
| `size()`                   | 获取联系人总数         |
| `searchByName(String)`     | 按姓名模糊查找         |
| `search(String)`           | 全局查找               |
| `saveToFile(String)`       | 保存到文件（对象流）    |
| `loadFromFile(String)`     | 从文件读取（对象流）    |

**对象流读写原理：**

保存时用 `ObjectOutputStream.writeObject()` 把整个 `ArrayList<Person>` 写进文件，读取时用 `ObjectInputStream.readObject()` 读回来。因为 `Person` 实现了 `Serializable`，所以可以整存整取。

### 3.3 MainFrame.java —— 主窗口

程序的主界面，继承了 `JFrame`。

**界面组成：**
- **菜单栏**：文件、编辑、查找三个菜单
- **表格**：用 `JTable` 显示联系人列表
- **右键菜单**：在表格上右键可以快捷操作

**菜单功能：**

| 菜单 | 菜单项       | 功能             |
| ---- | ------------ | ---------------- |
| 文件 | 新建         | 清空当前数据     |
|      | 打开...      | 打开 .dat 文件   |
|      | 保存         | 保存到当前文件   |
|      | 另存为...    | 选择路径保存     |
|      | 退出         | 关闭程序         |
| 编辑 | 添加联系人   | 弹出对话框添加   |
|      | 修改联系人   | 修改选中的联系人 |
|      | 删除联系人   | 删除选中的联系人 |
|      | 上移/下移    | 调整联系人顺序   |
|      | 更改类别     | 修改选中联系人的类别 |
| 查找 | 查找联系人   | 弹出查找对话框   |
|      | 显示全部     | 恢复显示全部数据 |

**右键菜单**会根据是否选中了联系人显示不同的内容：
- 没选中时：只显示"添加联系人"
- 选中时：显示添加、修改、删除、上移、下移、更改类别

**事件处理方式：** 使用 `ActionListener` 接口，在 `actionPerformed` 方法里通过 `e.getSource()` 判断点击了哪个菜单项。

### 3.4 PersonDialog.java —— 添加/修改对话框

继承 `JDialog`，用 `GridLayout` 布局。新增和编辑复用同一个窗口类。

包含四个输入框（类别、姓名、电话、邮件）和确定/取消按钮。点击确定时会检查姓名不能为空。

### 3.5 SearchDialog.java —— 查找对话框

提供两种搜索方式（下拉框选择）：
- 全局搜索：在姓名、电话、邮件、类别中查找
- 按姓名：只在姓名中查找

输入关键字后点击搜索，结果会显示在对话框内的表格中。

## 五、操作说明

### 编译运行

双击 `运行.bat`，或者命令行：

```
javac -d bin -encoding UTF-8 src/addressbook/*.java
java -cp bin addressbook.App
```

### 基本使用

1. **添加联系人**：菜单「编辑→添加联系人」，填写信息后确定
2. **修改联系人**：选中表格某一行，菜单「编辑→修改联系人」，或直接双击该行
3. **删除联系人**：选中后菜单「编辑→删除联系人」
4. **调整顺序**：选中后菜单「编辑→上移/下移」
5. **更改类别**：选中后菜单「编辑→更改类别」
6. **查找**：菜单「查找→查找联系人」，输入关键字搜索
7. **保存/打开**：通过「文件」菜单操作，文件扩展名为 .dat

### 右键操作

在表格上点右键可以快速完成添加、修改、删除、移动等操作，省去从菜单栏选择的步骤。

## 六、总结

本程序使用 Java 面向对象的方式完成了地址簿管理器。通过把功能拆分到不同的类中（Person 存数据、AddressBook 管逻辑、各个 Frame/Dialog 负责界面），代码结构清晰。序列化技术的使用让数据可以持久保存到文件中。
